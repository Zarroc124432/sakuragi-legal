Conditions d'utilisation — Sakuragi

En utilisant le bot Sakuragi, vous acceptez les conditions suivantes :

  • Le bot collecte uniquement les données nécessaires à son fonctionnement.
  • Aucune donnée personnelle n'est vendue ou partagée.
  • Les statistiques (messages / vocal) sont stockées uniquement pour l'affichage des commandes.
  • Les administrateurs du serveur sont responsables de l'utilisation du bot.
  • Le bot peut être modifié ou arrêté à tout moment.

Données stockées

  • ID Discord utilisateur
  • Statistiques d'activité
  • Logs de modération

Aucune donnée sensible n'est collectée.

Contact

Pour toute question, contactez l’administrateur du serveur utilisant Sakuragi.
